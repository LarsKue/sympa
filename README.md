# sympa
Embedding graphs in SYMetric sPAces

## Requirements
 - Python == 3.7 
 - Pytorch >= 1.4.0
 - Geoopt >= 0.1.2: install from repository is advised: ```pip install git+https://github.com/geoopt/geoopt.git```
 - networkx --> for preprocessing only
 - matplotlib --> for preprocessing only
 - tensorboardx
 - tqdm
